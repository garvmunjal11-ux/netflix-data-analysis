# Netflix Data Analysis

## 📌 Objective
Analyze Netflix dataset to find trends in movies and TV shows.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## 📊 Key Insights
- Movies are more than TV Shows
- Content increased after 2015
- USA produces most content

## 📈 Visualizations
- Bar charts
- Line graphs

## 📂 Dataset
Netflix dataset from Kaggle
